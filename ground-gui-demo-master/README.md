# Ground Station GUI Demo

## Technologies

-   JavaScript
    -   [Vue.js]("https://vuejs.org) and Vuex: front-end framework and client-side logic
    -   [Plotly.js]("https://plot.ly/javascript/): graphing library
-   CSS
    -   [Milligram]("https://milligram.io/"): front-end CSS framework
